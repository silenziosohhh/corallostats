Esempi

Snippet pronti: curl + header Authorization.

curl

curl "https://<host>/api/v1/summary" -H "Authorization: Bearer <API_KEY>"
curl "https://<host>/api/v1/stats/bedwars/Fed" -H "Authorization: Bearer <API_KEY>"

curl (più esempi)

curl "https://<host>/api/v1/clans" -H "Authorization: Bearer <API_KEY>"
curl "https://<host>/api/v1/clan-members/ExampleClan" -H "Authorization: Bearer <API_KEY>"
curl "https://<host>/api/v1/results?limit=50" -H "Authorization: Bearer <API_KEY>"

JavaScript (fetch)

const base = "https://<host>/api/v1";
const apiKey = process.env.CORALLO_API_KEY;

const res = await fetch(`${base}/summary`, {
  headers: { Authorization: `Bearer ${apiKey}` },
});
if (!res.ok) throw new Error(`HTTP ${res.status}`);
const data = await res.json();
console.log(data);

Python (requests)

import os, requests

base = "https://<host>/api/v1"
api_key = os.environ["CORALLO_API_KEY"]

r = requests.get(f"{base}/summary", headers={"Authorization": f"Bearer {api_key}"})
r.raise_for_status()
print(r.json())

Errori comuni

  • 401: header mancante (o key vuota).
  • 403: key errata o rigenerata.
  • 429: rate limit; rispetta Retry-After.
Suggerimento: aggiungi cache e retry progressivo lato client.
